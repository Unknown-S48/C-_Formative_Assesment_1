# Question 1

## Student Mark Calculator

### Technologies 
- C#

### Features 
This program takes the student name and 3 marks for 3 subjects then it displays the: name, total marks, average marks, checking if they passed with a rule set in place and then the currect time and date. Then you can press any key to exit the project.

### The Process
I had fun creating this I learned to build a C# project and had how to create a try catch without using a try catch. 

### How to Run
To run the program just navigate to the Question1 folder in the terminal then run 
- dotnet build
- dotnet run

# Question 2

## Simple ATM Console

### Technologies
- C#

### Features 
This program takes your name as an input and prints it out in upper case letters. You can also enter an account balance and then withdrawal an amount from that saved number. The program will loop and ask you to try again if your withdrawal amount is invalid acording to the set rules. It will also print Failed or Successful, then show the updated balance and currect transaction time. 

### The Process
I learned how to have to runable files in the same folder. I redid the same type of program loop allowing me to have you enter the incorrect information without stopping the program and doing it all over. This one was bit less fun to as Question 1 they didn't give much to one but a screenshot that I had to copy so I added some features like the looping and validations.

### How to Run
