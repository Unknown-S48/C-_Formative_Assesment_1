# Question 1

## Student Mark Calculator

### Technologies 
- C#

### Features 
This program takes the student name and 3 marks for 3 subjects then it displays the: name, total marks, average marks, checking if they passed with a rule set in place and then the currect time and date. Then you can press any key to exit the project.

### The Process
I had fun creating this I learned to build a C# project and had how to create a try catch without using a try catch. 

### How to Run
To run the program just navigate to the Question_1 file in the terminal then run 
- dotnet build
- dotnet run
